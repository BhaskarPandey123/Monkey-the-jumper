# Monkey-jump
A Monkey Go Happy-inspired browser game built with JavaScript and p5.js.

This project recreates the core mechanics of the original game while exploring collision detection, object spawning, and game state management. 

## Demo

Live Page:
https://bhaskarpandey123.github.io/Monkey-jump/

## How to Play
Open `index.html` in your browser - no installation needed.
Or clone it:
git clone https://github.com/BhaskarPandey123/Monkey-jump

## Features
- Banana collection system
- Stone obstacle generation
- Score system
- Game states (PLAY / END)
- Restart functionality

- p5.js
